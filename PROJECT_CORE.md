# 🚀 KINDER SPA BOT — CORE ARCHITECTURE

## 🎯 Цель проекта
Telegram-бот для записи клиентов в Kinder SPA с удобным UX, календарём и полной автоматизацией заявок.

---

## 🧩 Основные модули

### 1. USER FLOW (Клиент)
- /start
- Выбор услуги
- Выбор даты (календарь)
- Выбор времени
- Ввод данных (имя, телефон)
- Подтверждение записи
- Сохранение в Google Sheets

---

### 2. ADMIN FLOW (Администратор)
- Просмотр записей
- Фильтр по дате
- Изменение статуса:
  - NEW
  - CONFIRMED
  - DONE
  - CANCELLED

---

### 3. CALENDAR SYSTEM
- Инлайн-календарь
- Блокировка занятых слотов
- Работа с датами через utils/dates.py

---

### 4. DATA STORAGE
- Google Sheets (основное хранилище)
- Каждая запись:
  - ID
  - Имя
  - Телефон
  - Услуга
  - Дата
  - Время
  - Статус

---

### 5. REMINDERS
- Уведомление за 24 часа
- Уведомление за 2 часа

---

## ⚙️ ТЕКУЩИЕ ПРОБЛЕМЫ

- ❌ Нет единой модели записи
- ❌ Нет нормальной валидации данных
- ❌ Возможны дубли записей
- ❌ Календарь не проверяет занятость корректно
- ❌ Нет центра логики (размазано по файлам)
